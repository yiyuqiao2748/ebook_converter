# TXT to EPUB Converter - 电子书转换工具

一个功能强大的iOS应用，专门用于将TXT格式的电子书转换为标准EPUB格式，支持智能章节识别、自定义封面、批量转换等高级功能。

## ✨ 功能特性

- 📖 **智能章节识别** - 支持多种章节格式（中文"第X章"、英文"Chapter X"、数字编号等）
- 🎨 **自定义封面** - 支持上传自定义书籍封面图片，自动优化尺寸
- 📚 **批量转换** - 一次选择多个TXT文件批量转换
- 📜 **转换历史** - 自动保存转换记录，随时重新下载或分享
- 🌓 **深色模式** - 支持系统深色/浅色模式自动切换
- 🔄 **断点续传** - 大文件转换支持断点续传
- 📱 **iOS优化** - 针对iOS平台深度优化，完美支持iPhone和iPad
- 📤 **多种分享方式** - 支持AirDrop、邮件、保存到文件等多种导出方式

## 🔧 技术特点

- 完全符合EPUB 3.0标准，兼容所有主流电子书阅读器
- 自动优化书籍排版，支持自定义段落间距和字体样式
- 智能处理中文编码问题，自动识别GBK、UTF-8等编码
- 高效转换引擎，处理100MB以上大文件无压力
- 完善的错误处理机制，转换失败自动重试

## 🚀 快速开始

### 编译构建

```bash
# 获取依赖
flutter pub get

# iOS构建
flutter build ipa --release
```

### 开发调试

```bash
# 运行iOS模拟器
flutter run -d ios
```

## 📖 使用说明

1. 点击"选择TXT文件"按钮，选择要转换的电子书文件
2. （可选）上传自定义封面图片
3. 填写书籍信息（标题、作者）
4. 选择章节识别模式
5. 点击"开始转换"按钮等待完成
6. 转换完成后即可分享或保存EPUB文件

## 🔒 权限说明

应用需要以下权限：
- 存储访问权限：用于读取TXT文件和保存转换后的EPUB文件
- 照片访问权限：用于选择封面图片
